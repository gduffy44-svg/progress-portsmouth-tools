What is Housing Radar?

Progress Portsmouth monitors Portsmouth's housing landscape across four dimensions: development activity, zoning policy, market conditions, and government actions. We use AI-assisted research to scan public sources, then review and publish findings for residents, policymakers, and advocates.

This is not a live feed. Each category is scanned on a rolling basis and reviewed before publication. Results reflect conditions at the time of scanning. Findings older than 45 days are flagged for refresh.

Admin Access

Progress Portsmouth team only.

Incorrect password. Try again.
Workflow: (1) Click Scan All or Scan Now per category. (2) Review and edit each result. Add a source URL if you have one. (3) Click Publish — this updates the public view in this browser and prompts an export. (4) Commit and push the exported JSON to make it live for everyone. See the git steps below.
Publishing to the live site (do this after every session): After exporting, copy the JSON file into your repo at housing/housing-radar-data.json, then run:

git add housing/housing-radar-data.json
git commit -m "chore: update Housing Radar data [date]"
git push

Vercel deploys in ~30 seconds. Visitors will then load data from the server, not a browser cache.
⚠️ Security note: The admin password is stored in the HTML source and visible to anyone who inspects the file. This is editorial access control only — do not treat it as secure authentication.
